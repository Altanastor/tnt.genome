tnt.genome

tnt.board.genome is a special type of board that exposes the same methods explained in its API. TnT Genome also offers a few elements that are explained below. The most important are: three new features for genes, transcripts and genomic sequences, a modified location feature that includes the species and the chromosome being displayed in the board, a layout that avoids overlaps in genes and transcripts and integration with the Ensembl REST API to fetch genomic data.

{% highlight javascript linenos %} // Genome is a board. Location and axis tracks are included by default var genome = tnt.board.genome().species("human").gene("brca2").width(950); var gene_track = tnt.board.track() .height(200) .color("white") .display(tnt.board.track.feature.genome.gene() .color("#550055") ) .data(tnt.board.track.data.genome.gene()); var sequence_track = tnt.board.track() .height(30) .background_color("white") .display(tnt.board.track.feature.genome.sequence()) .data(tnt.board.track.data.genome.sequence() .limit(150) ); genome .zoom_in(100) .add_track(sequence_track) .add_track(gene_track); genome(document.getElementById("mydiv")); genome.start(); {% endhighlight %}

In the example above two tracks are created using the new data endpoints and displays: the first one is a gene track and the second one a sequence track. Although no location or axis tracks are created, they are included by default in the genome board. min and max are set automatically to 0 and the chromosome length automatically. The rest of the elements are the same explained before for TnT Board.
